# 🚀 Smart To-Do App

A simple and clean **To-Do Web Application** built using **HTML and CSS**.
This project focuses on creating a visually appealing and user-friendly interface for task management.

---

## 🌟 Features

* 📝 Add tasks (UI design)
* 🎯 Task layout with priority styling
* 📅 Deadline input field (UI)
* 🌙 Dark mode design (UI)
* 🎨 Modern UI using Tailwind CSS
* 📱 Responsive layout

---

## 🎨 UI Highlights

* Glassmorphism design
* Gradient background
* Clean and minimal layout
* Smooth hover effects

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Tailwind CSS)

## ⚙️ How to Run Locally

1. Download or clone the repository

```bash
git clone https://github.com/PREMPRATICK/smart-todo-app.git
```

2. Open the project folder

3. Run `index.html` in your browser
